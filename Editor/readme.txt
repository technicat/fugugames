Unity Editor scripts that add commands to a Fugu Games menu on the Unity menu bar.

ActivateRecursively and DeactivateRecursively calls SetActive on a hierarchy of GameObjects.
MeshInfo displays mesh stats on the selected GameObject
CreateChild creates a child GameObject of the selected GameObject
Build pops up a list of your per-target builds and lets you delete them
Scene pops up a list of scenes and lets you switch to one
